# Sali test Project

## Prerequisites

- Python 3.12
- pip

## Installation


1. Create virtualenv:

```bash
python -m venv venv
```
2. Activate venv

```bash
source venv/bin/activate
```

3. Install the required dependencies:

```bash
pip install -r requirements.txt
```

4. Run server

```bash
python app.py
```
